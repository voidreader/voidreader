### 안녕하세요! 이형진입니다. 👋

의료정보 SI 개발자 & 게임 개발자로 재직했으며,  
2017-23년 초까지 게임 스타트업을 창업하여 조직 운영 및 사내 프로젝트를 리드하였고,
지금은 평범한 직장인 개발자 A입니다.

유니티 기반의 클라이언트와 C#, NestJS, Node.js, MySQL, Redis, Socket.io 등을 활용한 백엔드 서버 개발을 주로 해왔습니다.
현재는 CLAUDE, Codex를 활용하여 프로젝트별 AI 개발을 위한 기반 작업 및 워크플로우 설계 및 운영에 집중하고 있습니다. 

3차 병원 SI, 모바일 게임, 스토리게임 플랫폼을 경험하며 게임 클라이언트부터 프론트엔드, 백엔드까지 폭넓게 개발해왔습니다.
